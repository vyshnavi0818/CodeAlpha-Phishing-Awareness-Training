# Phishing Awareness Training

## Project Overview

This project was created as part of the CodeAlpha Cyber Security Internship (Task 2).

The objective of this project is to educate users about phishing attacks, social engineering techniques, fake websites, and cybersecurity best practices.

## Topics Covered

* Introduction to Phishing
* Types of Phishing Attacks
* How to Identify Phishing Emails
* How to Identify Fake Websites
* Social Engineering Attacks
* Real-World Phishing Examples
* Best Practices to Stay Safe
* Interactive Quiz
* Key Takeaways
