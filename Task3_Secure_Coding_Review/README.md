# Secure Coding Review

## Project Overview

This project was created as part of the CodeAlpha Cyber Security Internship (Task 3).

The objective of this project is to review a Python Login System, identify security vulnerabilities, assess risks, and provide recommendations for secure coding practices.

## Files Included

* login_system.py
* Security_Review_Report.md
* Screenshots

## Vulnerabilities Identified

1. Hardcoded Credentials
2. Weak Password
3. No Login Attempt Limit
4. Plain Text Password Handling
